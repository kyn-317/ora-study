### [P2] C_06_019 B) — UNDO 테이블스페이스 OFFLINE 설명 보완
- **대상**: C_06_019 B) 오답 해설
- **우선순위**: P2
- **현재 내용**: "UNDO表領域はアクティブなトランザクションが存在する場合でもALTER TABLESPACE文で即座にオフラインに設定できる" (오답)
- **문제점**: UNDO 테이블스페이스는 현재 UNDO_TABLESPACE 파라미터로 지정된 활성 UNDO TS인 경우 직접 OFFLINE으로 전환할 수 없음. 해설에서 "다른 UNDO 테이블스페이스로 전환(ALTER SYSTEM SET UNDO_TABLESPACE)한 후 기존 UNDO 테이블스페이스를 삭제하는 절차"라고 했는데, OFFLINE과 삭제를 혼용. 좀 더 정확하게: 활성 UNDO TS를 OFFLINE으로 전환하는 것 자체가 불가하며, UNDO_TABLESPACE를 다른 TS로 전환 후 기존 TS의 ACTIVE 트랜잭션이 완료되어야 DROP이 가능하다는 설명이 더 정확.
- **수정 제안**: 해설을 "현재 활성 UNDO 테이블스페이스는 ALTER TABLESPACE ... OFFLINE으로 전환할 수 없습니다. 전환하려면 먼저 ALTER SYSTEM SET UNDO_TABLESPACE로 다른 UNDO 테이블스페이스를 지정해야 합니다."로 수정.
